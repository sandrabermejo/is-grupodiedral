\srsfuncion{Acceder a una oferta} \label{fun:accederoferta}
	Esta función muestra una oferta específica elegida por el cliente y ofrece la opción de compra.
	
\begin{enumerate}
	\item \textit{Prioridad}: media.
	\item \textit{Entradas}
	\begin{enumerate}
		\item Los datos detallados de la oferta estarn en el mismo idioma en el que esta el resumen de la oferta.
	\end{enumerate}
	\item \textit{Flujo de operaciones}
	\begin{enumerate}
		\item La oferta aparece descrita detalladamente especificando el/los artículo/s ofertados, por lo que si el usuario estuviese interesado en ella podrá comprar la oferta.
		\item Si la oferta consta de varias opciones de compra, el cliente puede elegir entre todas las que haya.
		\item Si el cliente escoge la opción \verb|Comprar|, se le redirecciona al proceso de \nameref{fun:iniciarpago}.
	\end{enumerate}
	\item \textit{Respuesta a situaciones no previstas}
	\begin{enumerate}
		\item Si al intentar acceder a una oferta el sistema falla, se mostrará un mensaje de error por pantalla informando de que los datos de esta oferta pueden ser que estén dañados o que simplemente no esté ya disponible. 
	\end{enumerate}
	\item \textit{Relación con otras funciones}\\
		Este caso es útil para \nameref{fun:mostrarofertas} y está relacionado con \nameref{fun:programaroferta}.
\end{enumerate}
